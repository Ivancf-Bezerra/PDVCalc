# Documentação do projeto

A pasta **`docs/`** concentra todos os arquivos `.md` usados para **regras** e **desenvolvimento** do projeto.

## Documentos

| Arquivo | Conteúdo |
|---------|----------|
| [DESENVOLVIMENTO.md](DESENVOLVIMENTO.md) | Regras, convenções, utilização da aplicação e guia de desenvolvimento (estrutura, comandos, novas features, Tailwind/Preline, testes). |

Novos documentos de regras, padrões ou desenvolvimento devem ser criados nesta pasta. O [README.md](../README.md) na raiz permanece como entrada principal do repositório.
